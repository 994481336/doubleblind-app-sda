# 飞行员双盲测试/复飞专项评估分析平台

这是一个基于 Streamlit 的评估数据分析平台，用于读取飞行员双盲测试和复飞专项评估 Excel 台账，统计飞行员得分、扣分项、评分项目失分、单位差异和个人明细，并导出报告。

## 设计思路

应用围绕“上传评估台账 -> 识别固定格式 -> 解析真实表头 -> 聚合评分记录 -> 输出分析报告”的流程设计。解析层优先保证双盲测试表和复飞专项表的字段、分值行、评分行、平均/合计行识别准确；分析层统一人员得分、扣分明细和单位统计口径；界面层提供解析预览、质量检查、筛选分析和会话内报告导出。

界面采用专业数据工作台风格：左侧为上传和筛选区，顶部为会话状态和页面导航，中部为解析质量、紧凑指标、小型图表和明细表。视觉上使用浅色液态毛玻璃面板、克制蓝灰色系统和高信息密度布局。

## 功能说明
- 支持上传多个 Excel 文件（每个文件代表一个公司或单位）
- 自动识别双盲测试表和复飞专项检查表
- 基于 Excel 真实表头识别扣分项，计算最终得分
- 多维度分析：单位对比、技术等级分析、科目分析、个人详情
- 可视化图表：得分分布、单位/技术等级对比、高频扣分项、科目失分、综合考评、风险指标等
- 导出 HTML 报告、Word 报告、CSV 数据和风险指标 Excel

## 当前支持格式

- 双盲测试表：分值行第 3 行，通常 93 个扣分列，字段为 `检查员`。
- 复飞专项检查表：分值行第 5 行，通常 149 个扣分列，字段为 `评估员`。
- 页面和导出会按当前数据范围显示原始称谓：仅双盲测试表显示 `检查员`，仅复飞专项检查表显示 `评估员`，混合上传时显示 `评分人员`。

## 快速开始

### 1. 安装 Python
使用 Python 3.10 或更高版本。

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 启动应用
```bash
streamlit run eastchina_doubleblindtestA.py
```

### 4. 运行测试
```bash
python3 -m unittest tests/test_parser.py
```

## 公开部署注意事项

如果部署为 Streamlit public app，请不要把真实 Excel、PDF、Word 报告、密钥或人员数据提交到公开项目文件中。当前项目已通过 `.gitignore` 排除本地 `files/` 样本目录。应用只在当前会话中处理上传文件，可下载结果，但不保存上传文件或分析历史。
